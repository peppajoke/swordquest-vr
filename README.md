# SwordQuestVR

A fast-paced VR sword fighting game built for Meta Quest. Fight waves of enemies with swords and guns in an arena-style combat experience.

## Playing

### VR Mode (Meta Quest)
Load the URL in your Meta Quest browser. Squeeze the grip triggers to spawn your swords. Hold both grip triggers together to rocket boost forward. Use the trigger buttons to fire guns.

### Desktop Mode
Open in a regular browser and click to capture the mouse.

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look |
| Space | Jump |
| Shift (hold) | Rocket boost |
| 1 | Equip sword |
| 2 | Equip guns |
| Scroll wheel | Switch weapon |
| Left click | Shoot (gun mode) |
| Right click | Swing sword (sword mode) |
| R | Reload |

## Enemies
- **Grunt** — basic melee attacker
- **Rifleman** — ranged attacker
- **Heavy** — slow but high damage
- **Assassin** — fast melee
- **Bomber** — explosive attacker
- **Drone** — aerial laser attacker
- **Wasp** — fast aerial stinger
- **Phoenix** — fire attacker, respawns
- **Boss** — high health, heavy damage

## Tech Stack
React Three Fiber, WebXR (@react-three/xr), Three.js, Express, TypeScript, Vite
